- 结果：主菜单、卡牌、方块、面板都有统一视觉语言。

## 第二轮修复

- 修复普通关失控判定：不再因为碎片数已达标而绕开失败。
- 修复同一行动重复支撑检查：一次行动最多执行一次阈值支撑检查，避免“不稳定”被跳过。
- 修复功能牌提前进入牌库：功能牌必须在奖励解锁后才加入后续牌库。
- 调整数值：普通关碎片目标提高，`level_01` 行动数下调，最终关行动数下调。
- 调整核心块：普通攻击不能直接命中核心块，最终关需要通过弱提示生成「核心裂解」。
