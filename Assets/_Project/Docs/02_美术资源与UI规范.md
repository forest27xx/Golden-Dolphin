日期：2026-06-20  
资源目录：`Assets/_Project/Resources/Art`

## 资源生成方式

当前所有资源由脚本生成：

`Assets/_Project/Tools/generate_visual_assets.py`

Agent 卡牌插画由以下脚本生成：

`Assets/_Project/Tools/generate_agent_card_art.py`

运行方式：

```powershell
python Assets/_Project/Tools/generate_visual_assets.py
```

生成后通过 Unity 菜单执行：

`Memory Tower / Configure Visual Assets`

该步骤会把 PNG 导入为 Sprite，并为按钮/面板设置九宫格边框。

## UI 布局原则

- 第一屏必须直接进入主菜单，不做营销页。
- 主菜单左侧留给标题和按钮，右侧展示危楼背景。
- 游戏场景左侧为建筑，右侧为操作信息，底部为手牌。
- 每张牌显示名称、伤害、坍塌值和一句效果；卡面背景优先使用 Agent 插画，选中态用描边而不是替换卡图。
- 方块上只保留 HP 与类型短标记，避免说明文字压住画面。

## 后续替换建议

- 主菜单背景可以替换为正式 2D key art。
- 卡牌底图可以保留当前结构，仅替换上方插画区。
- 方块图建议后续增加裂纹层级和坍塌帧动画。
- 中文字体需要在 Steam 版本前替换为可商用字体。
