# 《记忆危楼》Demo 变更记录

- 建立 `Assets/_Project` 目录结构。
- 新增 `MainMenu` 与 `Game` 场景。
- 新增卡牌、建筑、坍塌、奖励、存档、UI 等核心脚本。

- 新增 `generate_agent_card_art.py` 生成 8 张 Agent 卡牌插画。
- 新增主菜单背景、游戏背景、按钮、面板、卡牌底图、建筑方块、记忆碎片资源。

### 调整

- UI 从纯色块改为 Sprite 资源驱动。
- 方块由纯色按钮改为有材质和裂纹的状态图。
- 主菜单文案改为更贴近游戏主题。
- 普通关失控现在直接失败，避免碎片已达标但结构目标未完成时绕过失败。
- 每个行动最多触发一次支撑检查，保留“不稳定”预警阶段。
- 功能牌必须通过关卡奖励解锁后才进入后续牌库。
- 调整 `level_01`、`level_02`、`level_03` 和 `final` 的行动/碎片目标，让拆解规划更有存在感。
- 卡牌运行时优先加载 `Cards/AgentIllustrations` 下的竖版插画，选中态改为描边。
- 手牌布局从横向按钮感调整为竖卡比例。
- 最终关提示文案明确说明：红色核心只能由「听见回声」生成的「核心裂解」击破。
- README 当前限制已更新，不再误写为“所有美术都是 UI 占位色块”。
- Steam 草图改为官方尺寸：920x430、462x174、1232x706、748x896、1438x810、600x900、3840x1240、1280x720。
