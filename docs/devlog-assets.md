# 04 开发日志素材库

比赛日志需要平时积累，不要最后一天回忆。

## 每天至少收集

- 1 张截图或 1 段 GIF：展示今天的可运行变化。
- 1 条问题记录：哪怕是已解决的小问题。
- 1 条取舍记录：为什么今天不做某个功能。
- 1 条 Git 记录：分支名或关键 commit。

## 素材命名

```text
2026-06-18_player_move_first_pass.png
2026-06-20_build_test_windows.mp4
2026-06-29_main_menu_iteration.gif
```

## 说明模板

```text
日期：
素材：

当前状态：
是否可用于最终开发日志：
```
