# 03 会议记录与每日打卡

## 每日打卡格式

```text
【今日完成】
- xxx

【明日准备】
- xxx

【卡点】
- 无 / 卡在 xxx，需要谁帮忙

【对应任务】
- 飞书任务链接 / 任务名
```

## 今日任务摘要模板

```text
【今日任务摘要】6月X日

A：主任务 xxx，验收标准：xxx
B：主任务 xxx，验收标准：xxx
C：主任务 xxx，验收标准：xxx
D：主任务 xxx，验收标准：xxx
jun：跟进玩法文档 / Unity 辅助实现 / 验收 xxx

今天重点：先保证 xxx 能跑起来。
卡住直接在任务下面评论或群里说，不要等晚上。
```

## 问题记录模板

```text
【问题记录】
日期：6月X日
成员：xxx
环节：Unity 下载 / Git clone / 项目打开 / push / merge / build
现象：具体发生了什么，最好贴截图或报错全文
原因：如果已定位，写原因；没定位就写“未定位”
处理：最终怎么解决
是否影响其他人：是 / 否
需要写入规范：是 / 否
```
