## 2. 核心循环一句话
玩家从主菜单进入关卡，在有限出牌和弃牌资源内使用基础扑克牌形成牌型，满足建筑功能牌节点条件并抽出功能牌，累计坍塌值直到达成通关目标；通关后获得记忆碎片、功能牌解锁或跨关成长，并进入下一关、最终关或返回局外查看收藏内容。
## 3. 整体流程总览
整体流程分为三段：局外入口、普通关循环、最后一关流程。局外入口负责进入游戏和管理局外内容；普通关循环负责主要玩法推进；最后一关流程负责承接最终故事和终局通关。
## 4. 局外入口与菜单
主菜单是玩家进入游戏后的第一个交互界面，承担进入关卡和管理局外内容的功能。
- 新游戏：清空当前进度，并从开场流程或第一关开始。
- 继续游戏：读取已有进度，进入最近可继续的关卡或局外状态。
- 设置：调整音量、画面、震屏等基础选项。
- 收藏室：查看已解锁的功能牌、记忆碎片和相关文本内容。
- 制作人员：查看制作人员名单。
- 退出：退出游戏或返回标题态。

## 5. 关卡推进
普通关卡采用线性推进。玩家通关当前关卡后进入下一关；失败后可重新开始本关，也可以退出到主菜单。
- 通关当前关卡后，记录已通关进度。
- 通关奖励会影响后续关卡的可用内容。
- 失败不清空总进度，只重置当前关卡临时状态。
- 满足最终关开放条件后，玩家进入最后一关。
## 6. 普通关单局流程
普通关单局流程只描述骨架，详细规则以关联子系统为准。
```
开始
  → 初始化关卡
  → 初始化基础牌库与建筑功能牌节点
  → 发放初始手牌
  → 玩家出牌 / 弃牌 / 使用已抽出的功能牌
  → 结算本次操作结果
  → 判断胜利 / 失败 / 继续
  → 通关奖励 / 本关重开 / 退出到局外
```

## 9. 胜利、失败与重开
### 胜利
普通关胜利表示玩家在本关操作资源耗尽前达成关卡目标，并进入奖励结算。胜利后系统记录关卡进度，发放本关奖励，并判断是否进入下一关或最终关。
### 失败
失败表示玩家未能在当前关卡内完成目标，或当前关卡流程已无法继续。失败后玩家可以重新开始本关，也可以退出到主菜单。
### 重开
重开只重置当前关卡的临时状态，包括本局牌库顺序、手牌、已出牌、已弃牌、已抽出功能牌和本局结算进度。已完成的总进度、已解锁内容和设置项不因本关失败而清空。
## 10. 跨关奖励与成长
通关后的奖励和成长用于把单局结果带回长期流程，让玩家感到每一关都在推进整体进度。
- 记忆碎片：作为剧情和收藏内容的主要沉淀。
- 功能牌解锁：通关或达成条件后，新的功能牌进入后续可用范围。
- 跨关继承：部分已获得功能牌可在后续关卡以对应规则继续参与。
- 收藏室沉淀：已获得的记忆、功能牌和相关文本可在局外查看。
本文只描述跨关成长在核心循环中的作用，具体条件和继承配置以对应系统文档为准。

## 11. 最后一关流程
最后一关是核心循环的终局段，不只是普通关卡的数值放大版。它承接前面关卡获得的记忆、功能牌和剧情进度，进入最终故事节点。
```
进入最终故事节点
  → 初始化最终关
  → 玩家开始出牌
  → 满足条件后出现弱提示
  → 点击弱提示生成一次性功能牌
  → 使用一次性功能牌推进最终拆解
  → 判断最终胜利 / 失败
  → 播放结局或重开最终关
```
最终关可以保留弱提示按钮、一次性功能牌和最终结局等特殊流程。具体触发条件、一次性功能牌效果和结局分支不在本文展开。
