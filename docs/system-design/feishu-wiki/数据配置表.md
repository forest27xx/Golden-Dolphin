## 关卡与建筑节点配置补充（批次 F）
来源：卡牌系统 v0.2 revision 533、塔楼关卡系统 revision 57、存档系统 revision 109。以下字段是当前系统案建议，落地前仍需程序确认字段命名、字段类型、枚举值和数据格式。
### 配置边界
- LevelConfig 承载关卡级配置，例如目标坍塌值、基础牌库规则、出牌 / 弃牌次数、节点组引用和存档检查点规则。
- LevelFunctionNodeConfig 承载本关建筑上的功能牌节点配置，例如节点位置、节点角色、解锁条件、功能牌引用、抽出额外坍塌值和表现资源。
- LevelConfig / LevelFunctionNodeConfig 需要与 FunctionCardConfig / FunctionNodeConfig 建立引用关系，但不重复定义全部功能牌效果。功能牌具体效果仍由 FunctionEffectConfig 承载。
- GDD、旧 PRD、会议纪要等来源材料只作为参考；迁入配置前需经过保留 / 改写 / 废弃 / 待确认处理。
